# Demo for CPSC 304 Project

## 如何打开这个Project
- 用你们自己的Github账号(不是学校的)
- `git clone https://github.com/JohnLi1999/cpsc304_demo.git`

## 配置数据库
- 在MySQL WorkBench里创建一个`cpsc304_demo`数据库
    - 或者在任意地方运行这个SQL
        - `CREATE DATABASE cpsc304_demo`

## 如何运行这个Project
- 用intellij IDEA打开项目，点击运行键(下面图上的绿色按钮)
    ![Run](img/run.png)
- 或者用command line
    - ```
        cd cpsc304_demo
        ./gradlew bootRun 
      ```
